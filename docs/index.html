Overview

This repository contains the complete design models and figures for the passive soft base attaching gripper. All necessary files are provided to replicate the design and ensure reproducibility. The gripper is designed for robotic applications where passive attachment mechanisms are beneficial.

The repository includes:

  • Fusion 360 Files - Parametric models for design modification and adaptation
  • STL Files - Ready-to-print 3D models for prototyping and fabrication
  • Technical Figures - Detailed diagrams, views, and schematics
